# About GIMs configuration file

- We provide a user-editable configuration file, which allows users to customize the parameters of certain GIMs as needed. The default parameter settings are specified in the [Default Configuration File](tools_config.yml). The Default Configuration File contains the default parameter settings for LocusCompare2. If no adjustments are needed, users do not need to upload a configuration file.

- If users wish to modify additional parameters, they can refer to the list of adjustable parameters we have prepared for each GIM [Full Configuration File](tools_config_all_flexible_parameter.yml). For each parameter, users can simply add the desired value after the corresponding variable name, following the requirements of each tool. Note: Adding extra parameters that are not listed in the provided configuration may cause the program to return an error.
